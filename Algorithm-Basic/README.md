# 그림으로 배우는 알고리즘

### 알고리즘이란? 왜 배워야 하는가?
- 문제해결을 위한 처리절차
- 좋은 프로그램 구현을 위한 **훈련**

### 좋은 프로그램이란?
- 정당성(정확한 결과) + 정지성(정해진 시간)

### 알고리즘 종류

- 계산 : 최대공약수, 방정식, 정적분 ..등
- 정렬 : 단순 선택, 삽입, 병합, 교환, 셸, 퀵 등
- 검색 : 선형, 이진 등
- 매칭 : 단순문자열 일치. KMP, BM 등
